# pythonCollection（python代码集合）

**本项目目前仅在python2.7下实验，其他版本暂不维护，请灵活使用**

- 文件下载器
- pdf合并工具


## 文件下载器（downloader.py）

>从数据库或者文件中读取文件url地址，并且下载到本地

依赖库：

- requests
- progressbar
- MySQLdb（如果不从数据库读取数据则不需要）


## pdf合并工具（pdfmerger.py）

>合并pdf文件
